California School Directory

The California School Directory contains information about all California public schools, private schools, nonpublic nonsectarian schools, school districts, and county offices of education. The search feature below allows users to search for educational agencies in California by county, district, name, county-district-school (CDS) code, city, zip code, type, or status.

Search the California School Directory

Specify the search criteria in the fields below, and then select the "Search" button. For fields that accept typed input, a portion of the field value can be entered. For example, a search for a school name of "john" will result in all schools where "john" appears any place in the name.

Search Type


* Multi-select option using Shift or Control key + mouse click, or mouse dragging technique

Effective July 1, 2013, the California School Directory will no longer provide information for postsecondary districts or schools that are a part of the California Community College , California State University , or University of California systems. Postsecondary information should be obtained directly from the individual systems from July 1, 2013 forward.
